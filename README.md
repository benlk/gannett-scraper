# Gannett site comparisons

## Setup

1. [Install Nokogiri](http://www.nokogiri.org/tutorials/installing_nokogiri.html)
2. [Install PhantomJS](http://phantomjs.org/).


## Usage

```
chmod u+x run.sh
./run.sh
```
